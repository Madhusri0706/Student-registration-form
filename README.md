# Student Registration Form

A simple Student Registration Form built using HTML, CSS, and JavaScript.

## Features

* Add student details:

  * Name
  * Age
  * Gender
  * Course
  * Email
  * Password
* Email validation
* Password validation (minimum 6 characters)
* Save button disabled until valid input is entered
* Displays student details in a table
* Delete student records
* Form validation with error messages
* Responsive and attractive UI design
* Automatic form reset after saving

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Run

1. Download or clone the repository.
2. Open `index.html` in your browser.
3. Fill in the student details.
4. Click **Save** to add the record to the table.
5. Use **Delete** to remove a student record.

## Project Objective

This project demonstrates basic front-end development concepts including form handling, input validation, DOM manipulation, event handling, and dynamic table updates using JavaScript.
